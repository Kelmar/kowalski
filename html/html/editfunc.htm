F1 - Display general help, same as selecting Help - Contents from the menu bar.

Ctrl-F1 - Toggles the display of context-sensitive help when the cursor is on a 6502 instruction mnemonic or a recognized pseudo-op, same as selecting Help - Dynamic Help from the menu bar.

F2 - Jump to next bookmark.

Shift-F2 - Jump to previous bookmark.

Ctrl-F2 - Toggles bookmark on current text line.

Ctrl-Shift-F2 - Clear all bookmarks.

Ctrl-F - Find text.

Ctrl-G or F3 - Find next instance of text.

Shift-F3 - Find previous instance of text.

Ctrl-R - Find and replace text.

Alt-Insert - Toggle the highlighting of whitespace in text.
