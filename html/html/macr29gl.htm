.DBYTE or .DD

Directives that specify the byte pair values in the resulting code. The older byte is placed first.

Syntax:

[<label>[:]]     .DD expr { , expr }
 
[<label>[:]]     .DBYTE expr { , expr }

Example:

alpha:    .DD $1234, 0    ; will generate $12, $34,0, 0
 
beta:     .DBYTE alpha
 
             .DBYTE alpha-1, beta
