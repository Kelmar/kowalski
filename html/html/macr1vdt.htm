.DATE

Directive that stores the ASCII representation of the current date in memory. The format is "YYYY*MM*DD" where * can be 1 of 3 options. The string is not null terminated.

Syntax:

[<label>[:]]     .DATE    ;  uses default seperator YYYY-MM-DD
[<label>[:]]     .DATE 0  ;  uses no seperator YYYYMMDD
[<label>[:]]     .DATE '/';  uses custom seperator YYYY/MM/DD

Example:

alpha:    .DATE     ; will generate $32, $30, $32, $30, $2F, $31, $31, $2F, $32, $36 for Nov 26, 2020.
